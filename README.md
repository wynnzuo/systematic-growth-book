# 系统化成长：自我迭代方法论

> 一套可落地、可迭代的成长方法论，帮你把碎片经验变成系统能力。

[![Deploy mdBook](https://github.com/wynnzuo/systematic-growth-book/actions/workflows/deploy.yml/badge.svg)](https://github.com/wynnzuo/systematic-growth-book/actions/workflows/deploy.yml)

## 关于这本书

工作几年后我发现一个现象：很多人（包括我自己）每天都在学习、在忙，但年底回头看，真正沉淀下来的东西很少。碎片化的知识、即兴的决策、反复踩同样的坑——这不是能力问题，是缺少**系统**。

这本书记录和整理了我自己在成长过程中验证过的方法论。它们不是理论推演，而是在写代码、做项目、带团队、学新东西的过程中，被反复打磨出来的可操作框架。

## 目录结构

| 附录 | 参考书单、工具推荐 |

## 本地构建

需要安装 [mdBook](https://github.com/rust-lang/mdBook)：

```bash
# 安装 mdBook
curl -fsSL https://github.com/rust-lang/mdBook/releases/download/v0.4.40/mdbook-v0.4.40-x86_64-apple-darwin.tar.gz \
  | tar xzf - -C /usr/local/bin

# 构建并预览
mdbook serve --open
```
