# 项目管理：把人生当产品做

> 写代码需要架构，做事情也需要。

## 人生中处处是项目

- 换工作、学新技术、装修、健身——都是项目
- 用项目思维管理，而不是"顺其自然"

## 一个项目管理的简易框架

1. **定义目标**：完成的标准是什么？
2. **拆解任务**：WBS（工作分解结构）
3. **排优先级**：重要 vs 紧急
4. **执行追踪**：每周检查进度
5. **复盘归档**：经验入库

## 常用模板

```
项目名称：
目标（完成标准）：
截止时间：
关键里程碑：
- [ ] 里程碑 1：日期
- [ ] 里程碑 2：日期
复盘笔记：
```
