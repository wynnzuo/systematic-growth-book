# 复盘：从经验中提取价值

> 经验本身不是最好的老师，**复盘后的经验**才是。

## 复盘和总结的区别

| 总结 | 复盘 |
|------|------|
| 记录发生了什么 | 分析为什么会发生 |
| 一次性 | 可复用的方法论 |
| 关注结果 | 关注过程 + 决策链路 |

## KPT 复盘框架

- **Keep**（保持）：做对了什么，下次继续
- **Problem**（问题）：哪里没做好，原因是什么
- **Try**（尝试）：下一步打算怎么改

## 复盘的频率和形式

- 日复盘：10 分钟
- 周复盘：30 分钟
- 项目复盘：1-2 小时
