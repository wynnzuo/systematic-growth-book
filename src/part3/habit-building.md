# 习惯养成：让系统自动运行

> 不要依赖意志力，依赖系统。

## 习惯的运作原理

- 提示 → 渴求 → 反应 → 奖励
- 为什么改不掉坏习惯：环境 > 意志力

## 建立好习惯的四个原则

1. **让它显而易见**：把跑鞋放在门口
2. **让它有吸引力**：绑定你喜欢做的事
3. **让它简单易行**：两分钟法则
4. **让它令人愉悦**：即时奖励

## 习惯叠加

在 [现有习惯] 之后，我会 [新习惯]。

例子：喝完咖啡之后，我会写下今天的三个目标。

## 坏习惯的对抗

- 增加 friction（把手机放另一个房间）
- 改变环境（不在床上工作）
