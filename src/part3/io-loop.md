## Feynman 学习法

1. 选一个概念
2. 用最简单的语言解释给一个门外汉听
3. 发现卡壳的地方 → 回头学习
4. 简化、用类比

## 输出的最小闭环

- 读完一篇文章 → 写三句话总结
- 学完一个工具 → 写一个小 demo
- 做完一个项目 → 写一篇复盘
